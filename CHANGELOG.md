# 📝 AI Math Generator - 更新日誌

## v1.4 (2026-02-12)
### ✨ 新功能
- 新增 10 種應用題類型（來自 50_math_problems.md）
- 所有題目保持隨機生成，數字每次不同

### 📚 新增題目類型
| 類型 | 題目內容 | 運算 |
|------|---------|------|
| 大巴中巴座位 | 大巴限乘 X 人，中巴限乘 Y 人 | X×1+Y×3 |
| 買票問題 | 成人票 X 元，兒童票 Y 元 | X×大人+Y×小孩 |
| 還書問題 | 借走 X 本，分 Y 次還 | (總數-X)÷Y |
| 水果優惠 | 買 X 斤及以上，每斤優惠 Y 元 | 數量×優惠價 |
| 彩紙整花 | X 張紙做 Y 朵，分給 Z 個小朋友 | X×Y÷Z |
| 牛奶分發 | 每箱 X 盒，Y 班每班 Z 盒 | X×Y×Z |
| 水果分發 | X 班每班 Y 筐，每筐 Z 個 | X×Y×Z |
| 橡皮價錢 | 兩組買不同數量橡皮 | (數量1-數量2)×單價 |
| 船運人數 | 男生 X 船，女生 Y 船 | X×每船+Y×每船 |
| 電影院座位 | X 排每排 Y 座位，賣出 Z 張 | X×Y-Z |

## v1.3 (2026-02-09)
### ✨ 新功能
- 所有題目改為隨機生成，每次出題數字都唔同
- 新增9條應用題（乳酸菌、農場、鎮人口等）

### 📚 新增題目類型
| 類型 | 題目內容 |
|------|---------|
| 乳酸菌飲品 | 容量 ml × 打數 = 總毫升 |
| 農場牛奶 | 公升 × 單價 = 收入 |
| 魚蛋分俾同學 | 總數 ÷ 每串 ÷ 學生數 |
| 步行徑長度 | 兩段長度相加 |
| 牛奶重量 | 果汁 + 差額 = 牛奶 |
| 博物館人流 | 七月 + 八月 = 總人數 |
| 鎮人口比較 | A鎮 - B鎮 = 相差 |
| 志明零用錢 | 月錢 × 6個月 = 半年 |
| 超級市場果汁 | 三種果汁數量比較 |
