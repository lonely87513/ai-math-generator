# 📝 AI Math Generator - 更新日誌

## v1.3 (2026-02-09)
### ✨ 新功能
- 所有題目改為隨機生成，每次出題數字都唔同
- 新增9條應用題（乳酸菌、農場、鎮人口等）

### 📚 新增題目類型
| 類型 | 題目內容 |
|------|---------|
| 乳酸菌飲品 | 容量 ml × 打數 = 總毫升 |
| 農場牛奶 | 公升 × 單價 = 收入 |
| 魚蛋分俾同學 | 總數 ÷ 每串 ÷ 學生數 |
| 步行徑長度 | 兩段長度相加 |
| 牛奶重量 | 果汁 + 差額 = 牛奶 |
| 博物館人流 | 七月 + 八月 = 總人數 |
| 鎮人口比較 | A鎮 - B鎮 = 相差 |
| 志明零用錢 | 月錢 × 6個月 = 半年 |
| 超級市場果汁 | 三種果汁數量比較 |
